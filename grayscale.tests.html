Tests¶

Module contents¶

Tests package initialization module.

The tests package is designed to run tests of the functions of the main module of the GrayScale package.

This package can be run from the console, then the execution of all tests located in this package will be started.

The modules of this package import pytest and unittest modules and functions from os, pathlib and doctest modules.

Content:

Modules:

  • __init__ - package initialization

  • __main__ - running all package tests

  • doctests - doctest tests

  • test_pytest - pytest tests

  • test_unittest - unittest tests

True

True

True

>>> from grayscale.shades import to_gray

(0, 0, 0)
